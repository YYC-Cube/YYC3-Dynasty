# 0105-YYC3-根基架构-鱼符执行凭证

> **版本**：v1.0.0 ｜ **日期**：2026.06.14 ｜ **来源**：Dynasty Framework 第一章 1.5
>
> 尚书省派发任务时同步下发对应部门令牌，任务办结后令牌缴回。

---

## 1.1 六部鱼符功能矩阵

| 令牌 | 对应功能 | 持有部门 | 资源类型 |
|------|---------|----------|----------|
| 户部鱼符 | 数据调用权限、算力配额、存储资源 | 户部 | 数据/算力/存储 |
| 礼部鱼符 | 规范发布、文档生成、知识库编辑 | 礼部 | 文档/知识库 |
| 兵部鱼符 | 代码仓库读写、部署权限、Bug修复 | 兵部 | 代码/部署 |
| 刑部鱼符 | 安全扫描、审计权限、风险处置 | 刑部 | 安全/审计 |
| 工部鱼符 | CI/CD流水线、服务器操作、工具链配置 | 工部 | 运维/工具 |
| 吏部鱼符 | Agent管理、权限分配、考核奖惩 | 吏部 | 人员/权限 |

---

## 1.2 核心规则

### 规则一：派发与缴回
- 尚书省派发任务时同步下发鱼符，办结后缴回注销

### 规则二：并发控制
- 每个部门令牌数量 = 并发任务上限，用尽自动排队

### 规则三：跨部门借用
- 令牌可跨部门借用，需尚书省审批，逾期自动预警

### 生命周期状态机

```
┌─────────┐    派发    ┌─────────┐    办结    ┌─────────┐
│  库存   │ ───────→ │  持有中  │ ───────→ │  已缴回  │
│ (空闲)  │          │ (使用中) │          │ (归档)  │
└─────────┘          └────┬────┘          └─────────┘
                         │
                    超时预警
                         │
                         ▼
                    ┌─────────┐
                    │  逾期   │
                    │ (异常)  │
                    └─────────┘
```
