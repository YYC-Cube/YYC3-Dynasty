# 0104-YYC3-根基架构-竹箴规则知识库

> **版本**：v1.0.0 ｜ **日期**：2026.06.14 ｜ **来源**：Dynasty Framework 第一章 1.4
>
> 对应隋唐"律令格式"四元法律体系，为门下省审议唯一依据。

---

## 1.1 律令格式体系

| 分类 | 对应规则 | 示例 | 系统映射 |
|------|---------|------|----------|
| **律** | 系统红线规则、安全禁令、硬性约束 | 不可绕过审批、不可越权操作 | `SecurityPolicy` / `AccessControl` |
| **令** | 部门职能规范、权责边界、流程标准 | 中书不得越权审议、兵部不得调用户部数据 | `DepartmentSpec` / `WorkflowRules` |
| **格** | 奖惩细则、考核标准、权限配额 | 连续3次封驳该部门扣分、令牌上限=部门等级 | `ScoringRules` / `QuotaManagement` |
| **式** | 操作规范、输出格式、接口标准 | API返回格式、日志记录规范 | `APISpec` / `LogFormat` / `OutputTemplate` |

---

## 1.2 UI规则
