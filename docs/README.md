## 文档体系概览

本文档体系遵循 **YYC³ 团队规范-开发标准**，将 Dynasty Framework 的完整设计规范按 **六大类别** 进行结构化拆分，确保文档的可维护性、可追溯性与一致性。

### 命名规范

```
目录命名：{序号}-YYC3-古典文化-{四个字的大类名称}
文件命名：{序号}{序号}-YYC3-{大类名称}-{具体内容}
```
