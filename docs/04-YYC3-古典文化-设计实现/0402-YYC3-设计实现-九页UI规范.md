## 四、各页面详细规格

### P1 欢迎启动页 · 玉玺启宫门

**角色定位**：隋唐宫廷礼制顾问兼 UI 视觉设计师

**布局结构**：
```
┌─────────────────────────────────────┐
│           [玉玺图标 56px]           │
│      YYC³ Dynasty (楷体 28px 金)    │
│   三省以治 · 六部以行 (14px 次级)   │
│                                     │
│  ┌──────┐ ┌──────┐ ┌──────┐        │
│  │朝堂听政│ │王朝典章│ │勋绩考课│     │
│  └──────┘ └──────┘ └──────┘        │
│  ┌──────┐ ┌──────┐ ┌──────┐        │
│  │敕令流转│ │太史监候│ │宫阙规制│     │
│  └──────┘ └──────┘ └──────┘        │
│                                     │
│  ══════════════════════════════    │
│  紫微城中枢 · 十二朝臣当值          │
│       [奉天承运] (鎏金渐变按钮)     │
└─────────────────────────────────────┘
```

**启动动效**：玉玺落下→印泥扩散→内容渐显（3帧）

### P2 朝堂大厅 · 紫微中轴线 ⭐ 核心页

**角色定位**：古建筑可视化专家兼数据看板设计师

**节点列表**：
1. 天堂（天子）- 通栏大卡 - 待御批数/全局健康度
2. 明堂（储君）- 通栏中卡 - 待分拣数/监国状态
3. 应天门·中书 - 三栏左 - 待草拟数/方案库命中率
4. 应天门·门下 - 三栏中 - 待审议数/今日封驳次数
5. 应天门·尚书 - 三栏右 - 待派发数/令牌余量
6. 天津桥·六部 - 左右两列 - 各部门任务数/剩余令牌
7. 端门（早朝）- 通栏 - 今日简报摘要
8. 定鼎门（归档）- 通栏大卡 - 办结率/平均耗时/通过率/异常数

**特色组件**：
- 天子驾六微型仪表盘（六马健康状态）
- 六阶段流转示意条

### P3 十三王朝时间轴 · 洛邑千年 ⭐ 技能库核心页

**角色定位**：历史学家兼信息架构设计师

**核心组件**：
- 13个朝代圆形选择器（横向排布）
- 7个类别筛选Tag组
- 26张技能卡片（Component + Variants）
- 竹简样式详情面板（Overlay）

### P4-P9 规格摘要

| 页面 | 核心组件 | 特色交互 |
|------|----------|----------|
| P4 勋章墙 | 17枚勋章网格 | 解锁动效/星级纹样 |
| P5 旨意板 | 七阶步骤条 | 流转拖拽/封驳回流 |
| P6 Hub浮窗 | 双态切换(收起/展开) | 密奏通道/快捷操作 |
| P7 竹箴典章 | 律令格式四分类 | 条文搜索/违规高亮 |
| P8 太史监候 | 实时指标仪表盘 | 健康度/异常告警 |
| P9 宫阙规制 | 配置表单 | 主题切换/权限管理 |
