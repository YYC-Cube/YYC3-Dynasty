---

## 一、双层技能协同模型

### 架构总览

```
┌─────────────────────────────────────────────┐
│           双层技能协同模型                    │
│                                             │
│  L1 文化层 — 朝代 × 治世技能                  │
│  ├── 触发: 用户在时间轴选择/解锁朝代           │
│  ├── 效果: 为系统提供文化属性加成              │
│  │   例: 选择"唐"-> 桃李满天下(主动)激活       │
│  │       -> 所有 Agent 回复质量 +15%           │
│  │       -> 国色天香(被动) -> UI 金碧辉煌主题    │
│  └── 持续: 当前会话 / 直到切换朝代             │
│                                             │
│  L2 职能层 — 角色 × 业务技能                  │
│  ├── 触发: Agent 在流程中按条件自动触发         │
│  ├── 效果: 直接影响敕令流转结果                │
│  │   例: 门下省触发"执法如山"(被动)             │
│  │       -> 自动高亮违规项，建议封驳             │
│  │   例: 中书省触发"龙章凤藻"(主动)             │
│  │       -> 自动生成完整执行方案                │
│  └── 冷却: 政令值资源单位                      │
│                                             │
│  协同规则:                                   │
│  L1 朝代加成 x L2 技能基础效果 = 最终效果        │
│  例: "唐"朝代加成(x1.15) x 中书龙章凤藻         │
│    = 方案质量评分 x 1.15                       │
└─────────────────────────────────────────────┘
```

## 二、技能卡片交互状态

| 状态 | 视觉表现 | 可执行操作 |
|------|---------|-----------|
| **未解锁** | 灰色卡片 + 锁图标 + "解锁条件: XXX" | 查看条件提示 |
| **已解锁·未激活** | 正常卡片 + "激活"开关(关闭态) | 点击激活 |
| **已解锁·已激活** | 金色边框呼吸光效 + "激活"开关(开启态) | 点击停用 / 展开详情 |
| **冷却中** | 卡片半透明 + 倒计时环 + "冷却: 02:30" | 等待恢复 |
| **被动常驻** | 绿色左竖线标识 + "被动"标签 | 无法手动触发，自动生效 |

## 三、七步创建闭环流程

```
[1] 表单填写 → [2] 实时验证 → [3] 效果预览 → [4] 权限检查 
    → [5] 确认弹窗 → [6] 异步创建 → [7] 创建后处理
```

### 表单验证规则矩阵

| 字段 | 验证类型 | 规则 | 错误提示 |
|------|---------|------|---------|
| 技能名称 | 格式+唯一性 | 2-12字中文/英文，不可重名 | "技能名已存在" |
| 所属层级 | 枚举 | L1/L2 二选一 | -- |
| 关联朝代 | 条件必填 | L1必须选择朝代 | "L1文化层技能必须关联朝代" |
| 技能类型 | 枚举 | 主动/被动 二选一 | -- |
| 类别 | 枚举 | 必须在8大类别中选择 | "请选择技能类别" |
| 加成系数 | 数值范围 | 被动技能专用，0.5-2.0 | "系数范围0.5-2.0" |

## 四、事件总线集成

### 跨页面事件通信矩阵

| 事件名 | 触发源 | 监听方 | 数据载荷 |
|--------|-------|--------|---------|
| `skill:activated` | P3时间轴 | P2大厅/P5旨意板/P8监控 | `{skillId, layer, dynasty}` |
| `skill:deactivated` | P3时间轴 | 全局 | `{skillId, reason}` |
| `skill:triggered` | P5旨意板 | P3时间轴/P8监控 | `{skillId, edictId, result}` |
| `skill:created` | P3时间轴 | 全局 | `{skillInfo}` |
| `dynasty:changed` | P3时间轴 | 全局 | `{fromDynasty, toDynasty}` |
