## 一、产品概述

### 核心理念

**"三省以治，六部以行，技艺相融，行之有效"**

将洛阳十三朝古都的文化底蕴与现代多Agent协作系统深度融合，打造一套具有东方美学特质的智能工作台。

### 视觉关键词

| 关键词 | 英文 | 应用场景 |
|--------|------|----------|
| 宣纸肌理 | Paper Texture | 页面背景底纹 |
| 卷轴展开 | Scroll Unfold | 面板/抽屉动画 |
| 玉玺钤印 | Seal Stamping | 确认/生效操作 |
| 鎏金点缀 | Gold Accent | 高亮/激活状态 |
| 印章朱红 | Vermilion Red | 重要操作/警告 |
| 竹简竖排 | Bamboo Vertical | 典章/规则展示 |

## 二、设计系统

### 2.1 色彩体系

#### 主色调（古风色板）

| 色彩名称 | 色值 | CSS变量 | 用途 |
|----------|------|---------|------|
| 宣纸底色 | #1E180E | `--bg-paper` | 主背景 |
| 鎏金主色 | #C9A96E | `--gold-primary` | 主色调/高亮 |
| 鎏金浅色 | rgba(201,169,110,0.15) | `--gold-subtle` | 卡片背景/悬浮 |
| 印章朱红 | #8B2323 | `--vermilion` | 重要操作/印章 |
| 竹青色 | #4A7C59 | `--bamboo-green` | 成功状态/通过 |
| 青铜绿 | #5B7553 | `--bronze` | 正常状态/信息 |

#### 功能色

| 状态 | 色值 | 用途 |
|------|------|------|
| Success | #4A7C59 | 通过/成功/正常 |
| Warning | #D4A017 | 警告/待处理 |
| Error | #8B2323 | 错误/封驳/异常 |
| Info | #5B7553 | 信息/默认 |

### 2.2 字体规范

| 层级 | 字号 | 字重 | 字体 | 行高 | 用途 |
|------|------|------|------|------|------|
| H1 标题 | 28px | 700 | 楷体/KaiTi | 1.4 | 页面主标题 |
| H2 章节 | 22px | 600 | 楷体 | 1.4 | 区块标题 |
| H3 模块 | 18px | 600 | 宋体/SimSun | 1.5 | 组件标题 |
| Body 正文 | 14px | 400 | 宋体 | 1.6 | 正文内容 |
| Caption 辅助 | 12px | 400 | 宋体 | 1.5 | 辅助说明 |
| Code 代码 | 13px | 400 | Source Code Pro | 1.5 | 代码/数据 |

### 2.3 间距 & 圆角

| Token | 值 | 用途 |
|-------|-----|------|
| `--space-xs` | 4px | 图标间距 |
| `--space-sm` | 8px | 元素内边距 |
| `--space-md` | 16px | 组件间距 |
| `--space-lg` | 24px | 区块间距 |
| `--space-xl` | 32px | 大区块间距 |
| `--radius-sm` | 4px | 小元素圆角 |
| `--radius-md` | 8px | 卡片/按钮圆角 |
| `--radius-lg` | 12px | 大面板圆角 |

### 2.4 阴影层级

| 层级 | 值 | 用途 |
|------|-----|------|
| `--shadow-sm` | `0 1px 2px rgba(0,0,0,0.3)` | 悬浮微抬升 |
| `--shadow-md` | `0 4px 12px rgba(0,0,0,0.4)` | 弹窗/卡片 |
| `--shadow-lg` | `0 8px 24px rgba(0,0,0,0.5)` | 模态框 |
| `--shadow-gold` | `0 0 20px rgba(201,169,110,0.3)` | 激活态光晕 |

### 2.5 边框样式

| 样式 | 值 | 用途 |
|------|-----|------|
| 默认边框 | `1px solid rgba(201,169,110,0.2)` | 分隔线/卡片边框 |
| 激活边框 | `1px solid #C9A96E` | 选中/聚焦态 |
| 封驳边框 | `2px solid #8B2323` | 异常/驳回态 |
| 虚线分隔 | `1px dashed rgba(201,169,110,0.15)` | 区域分组 |
