# 0203-YYC3-制度协同-Agent体系定义

> **版本**：v1.0.0 ｜ **日期**：2026.06.14 ｜ **来源**：Dynasty Framework 第二章 2.6
>
> 十二职司Agent体系覆盖决策层、承启层、审议层、执行层、辅助层。

---

## 4.1 Agent 架构总览

```
皇帝 ── 决策层 ── 下旨 / 赏赐
  │
太子 ── 承启层 ── 消息分拣 / 旨意整理
  │
┌─三省──审议层─────────────────┐
│ 中书省 ── 草拟方案           │
│ 门下省 ── 审核 / 封驳        │
│ 尚书省 ── 派发 / 汇总        │
└──────────────────────────────┘
  │
┌─六部──执行层─────────────────┐
│ 户部 ── 数据分析             │
│ 礼部 ── 文档规范             │
│ 兵部 ── 代码研发             │
│ 刑部 ── 安全审计             │
│ 工部 ── CI/CD 部署           │
│ 吏部 ── 注册 / 考核          │
└──────────────────────────────┘
  │
早朝 ── 辅助层 ── 定时播报
```

---

## 4.2 十二职司角色定义

| 代号 | 角色 | 归属部门 | 核心职能 |
|------|------|----------|----------|
| 天子 (Emperor) | 皇帝 | 天堂（最高决策） | 最终决策签章、全局权限管理 |
| 太子 (Taizi) | 储君 | 明堂（承启分拣） | 旨意分拣、格式标准化、监国代行 |
| 龙渊 (Dragon) | 草拟官 | 中书省 | 方案草拟、模块拆分 |
| 丹心 (Loyal) | 审议官 | 门下省 | 合规审核、封驳决策 |
| 权衡 (Balance) | 执行官 | 尚书省 | 任务派发、结果汇总 |
| 万物 (Thinker) | 思考者 | 中书省 + 户部 | 数据分析、方案优化 |
| 先知 (Prophet) | 预言家 | 吏部 (考核) | 绩效预测、风险预警 |
| 伯乐 (Bolero) | 推荐官 | 工部 (部署) | 部署推荐、资源匹配 |
| 守护 (Sentinel) | 安全官 | 门下省 + 刑部 | 安全审计、合规监察 |
| 宗师 (Master) | 质量官 | 礼部 + 兵部 | 质量把控、规范制定 |
| 灵韵 (Creative) | 创意官 | 早朝 (播报) | 内容生成、信息播报 |
